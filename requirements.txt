#   - All versions are pinned (==) to guarantee reproducible environments.

#   - Graph library omitted intentionally. The FacilityGraph (src/spatial.py)
#     is a pure-Python adjacency dict implementation, demonstrating explicit
#     command of graph data structures — a deliberate academic decision.
#
#   - Linting tools (e.g. pep8 checkers) are development-only and are
#     excluded from this runtime requirements file. Install them separately
#     in development environments only.
#
#   - torch and torchvision are listed but installation should follow the
#     official PyTorch matrix at https://pytorch.org/get-started/locally/
#     to match the local CUDA version (tested on CUDA 11.8 / RTX 3060).
#     Override with the appropriate +cu118 index URL if needed:
#       pip install torch==2.2.1+cu118 torchvision==0.17.1+cu118 \
#                   --index-url https://download.pytorch.org/whl/cu118
